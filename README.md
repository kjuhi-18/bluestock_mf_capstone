# 🌟 Project Highlights

This capstone project simulates a complete Mutual Fund Analytics Platform used by asset management companies, analysts, and investors.

The platform performs:

* ETL Pipeline
* Data Cleaning
* Exploratory Data Analysis
* Performance Analytics
* Advanced Risk Analytics
* Power BI Dashboard
* Monte Carlo Simulation
* Automated HTML Reporting

## Project Objectives

1. Build ETL pipelines for mutual fund data.
2. Clean and validate datasets.
3. Perform EDA and generate insights.
4. Compute performance metrics.
5. Analyze risk and investor behaviour.
6. Build interactive dashboards.
7. Develop advanced analytics modules.
8. Deliver business recommendations.

## Tech Stack

* Python

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/kjuhi-18/bluestock_mf_capstone
cd Bluestock_MF_Capstone
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run ETL Pipeline

```bash
python scripts/etl_pipeline.py
```

## Run Recommendation Engine

```bash
python scripts/recommender.py
```

## Generate Weekly Report

```bash
python B5_automation/email_report.py
```

## Open Dashboard

Open:

```text
dashboard/bluestock_mf_dashboard.pbix
```

using Power BI Desktop.
